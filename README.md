## üöÄ Overview
**ZubairDevCoder Blog** is a modern **Full Stack Blog platform** built with **Next.js, React, TailwindCSS, Firebase, Shadcn UI, and Framer Motion**.  

It provides:
- **Clean SaaS-style UI**  
- **Mobile responsive blog cards**  
- **Admin panel for managing posts & categories**  
- **SEO-friendly structure**  

Perfect for **developers learning modern web development** or building **portfolio/blog websites**.
